# Otimização de funções

A aula separa a definição da função objetivo da escolha do método. O exemplo usa uma função de custo com quantidade produzida, custo fixo, custo variável e penalidade por afastamento da demanda alvo.

Métodos abordados:

- Busca em grade
- Busca ternária
- Descida do gradiente

```bash
python examples/otimizacao_custos.py
```

O laboratório usa apenas a biblioteca padrão do Python.
