# Solvers e otimizadores

Este bloco foi separado em quatro módulos independentes:

- [Dijkstra](../dijkstra/) - menor caminho em grafos ponderados com custos não negativos.
- [A*](../a-star/) - busca heurística com `f(n)=g(n)+h(n)`.
- [Algoritmo genético](../algoritmo-genetico/) - busca evolutiva em espaços discretos.
- [PSO](../pso/) - Particle Swarm Optimization para espaços contínuos.

Os novos módulos trazem formulação matemática, exemplos guiados, simulações interativas e laboratórios Python próprios.
