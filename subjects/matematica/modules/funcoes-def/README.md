# Funções com def

A aula conecta a ideia matemática de função ao uso de `def` em Python: entrada, regra, retorno, parâmetros, validação, composição e função de custo.

```bash
python examples/funcoes_custo.py
```

O laboratório usa apenas a biblioteca padrão e imprime cenários de custo, lucro e erro quadrático médio.
