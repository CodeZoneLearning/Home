01MODELO

// OBJETO, TIPO E VALOR

Python carrega o tipo junto com o objeto.

Uma variável é um nome que referencia um objeto. O objeto tem identidade, tipo e valor. A variável pode passar a apontar para outro objeto, mas o tipo do objeto não muda.

NAMEtemperature

Um nome no namespace atual.

→
OBJECT68.4

Um valor existente na memória.

→
TYPEfloat

Operações e comportamento disponíveis.

TYPE()

Descobrir o tipo exato

Útil para inspeção e aprendizado. Em regras de negócio, normalmente prefira verificar comportamento ou usar `isinstance`.

type(68.4) is float
ISINSTANCE()

Verificar uma família de tipos

Considera herança. Atenção: `bool` é uma subclasse de `int`, então regras numéricas podem precisar excluí-lo.

isinstance(True, int) # True
TIPAGEM DINÂMICA

O tipo pertence ao objeto em tempo de execução. Isso não significa ausência de regras: operações incompatíveis ainda produzem erros claros.

02ESCALARES

// UM VALOR POR OBJETO

Números, texto, verdade e ausência.

Escalares representam unidades de informação. Escolher entre eles define precisão, operações e significado.

INT2

Inteiros

Contagens, índices e classes discretas, sem parte decimal.

health_status = 2
FLOAT0.91

Ponto flutuante

Medições e aproximações. Nem todo decimal é representado exatamente em binário.

confidence = 0.91
STR"P-14"

Strings

Texto Unicode imutável. O tipo se chama `str`, não `string`.

equipment_id = "P-14"
BOOL / NONETrue

Estado e ausência

`bool` expressa condições; `None` representa intencionalmente nenhum valor.

active = True
FLOAT NÃO É DINHEIRO

Para valores financeiros ou casas decimais exatas, considere `decimal.Decimal`. Para medições, documente tolerâncias ao comparar.

04PRECISÃO

// NOMES PARECIDOS, CONCEITOS DIFERENTES

Docstring não é tipo.
Matriz também não.

Uma docstring é uma `str` posicionada no início de um módulo, classe ou função e exposta em `__doc__`. Já uma matriz é um conceito matemático que precisa de uma representação.

DOCSTRING

Documentação acessível pelo programa

def transpose(matrix):
    """Return the transposed matrix."""
    ...

print(transpose.__doc__)

Comentários explicam decisões para leitores do código. Docstrings documentam a interface de objetos Python.

MATRIX

Lista de listas ou array especializado

matrix = [
    [68.2, 4.1, 0.32],
    [69.0, 4.4, 0.35],
]

Listas aninhadas servem para aprendizado e pequenos dados. Cálculo numérico real normalmente pede `numpy.ndarray`.

MATRIX_LAB · 2 × 3

Altere a leitura da matriz.

Linhas representam testes; colunas representam temperatura, vibração e pressão.

2 testes × 3 sensores

RETANGULARIDADE É UMA REGRA

Uma lista de listas pode ter linhas de tamanhos diferentes. Se o domínio exige matriz, valide essa invariável antes de calcular.
