# Prompts e saídas estruturadas

Esta aula abre o subject de Generative AI com foco em prompts como contratos operacionais.

## Objetivo

Ao terminar, a pessoa deve conseguir:

- diferenciar um pedido genérico de uma especificação de tarefa;
- escrever prompts com papel, tarefa, contexto, restrições e formato de saída;
- usar exemplos e contraexemplos para reduzir ambiguidade;
- definir saídas em JSON para integração com outros sistemas;
- entender que prompt reduz erro, mas validação protege o produto.

## Prática recomendada

Para cada prompt importante, mantenha:

- versão do prompt;
- exemplos de entrada;
- saída esperada;
- casos que devem falhar;
- schema de resposta;
- critérios de aprovação.
