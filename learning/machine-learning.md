# Machine Learning: workflow de modelos

1. Defina a métrica ligada ao problema.
2. Separe treino, validação e teste antes de explorar.
3. Construa um baseline simples.
4. Crie um pipeline reproduzível.
5. Analise erros, não apenas a média.

## Avaliação
- [ ] Comparação com baseline
- [ ] Ausência de vazamento de dados
- [ ] Métrica por segmento relevante
- [ ] Latência e custo medidos
- [ ] Limitações documentadas
