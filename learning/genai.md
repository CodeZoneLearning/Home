# Generative AI: padrões de aplicações

- Prompt estruturado para tarefas simples
- RAG para conhecimento externo verificável
- Tool calling para ações e dados atuais
- Agentes apenas quando o fluxo exige decisões iterativas

## Avaliação
Crie um conjunto fixo de casos, defina critérios antes dos testes e meça qualidade, latência, custo e segurança separadamente.
