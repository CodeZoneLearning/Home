# Data Science: roadmap de análise

1. Formular uma pergunta verificável.
2. Identificar a origem e as limitações dos dados.
3. Limpar com transformações reproduzíveis.
4. Explorar distribuições e relações.
5. Comunicar conclusão, incerteza e próximo passo.

## Projeto-guia
Escolha um conjunto de dados público, produza um notebook limpo, três visualizações e um README que explique decisões e limitações.
