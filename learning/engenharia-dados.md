# Engenharia de Dados: pipeline do zero

## Fluxo
Fonte → ingestão → validação → transformação → armazenamento → consumo.

## Arquitetura
- Contrato de dados explícito
- Processamento idempotente
- Particionamento coerente com o acesso
- Testes de qualidade e observabilidade
- Documentação de execução e recuperação
